    path('admin/', admin.site.urls),

]

# Evitar código repetitivo por meio do DRY (Dont Repeat Yourself)

Criar um arquivo base.html em templates com o trecho repetido de código

usar chamadas {% block content %}{% endblock %} indicando a localização dos blocos que serão adicionados de outros arquivos.

Neste outros arquivos usar chamadas {% extends 'galeria/base.html' %}, {% block content %} e {% endblock %}



Quando temos regiões repetidas com botões, rodapés, etc, chamamos de partials. 

Criar uma pasta partials dentro de templates. Dentro dela criar um arquivo _footer.html

no local que queremos adicionar, usar a tag {% include 'galeria/partials/_footer.html' %}
